# AI Content Detection Tool 🔍

Created in less than 3 hours at T126 No-Code Hackathon 
King's Own Institute, Sydney – May 2026 

## What it does
Helps students to submit their project work and get
an all-round analysis of the content, ensuring that there is no use of AI-generated content in their work.

## How it was created
- Created by using ChatGPT/GPT-4 for analysis logic
- Development of a web application without code
- Accomplished in a 3-hour hackathon challenge
